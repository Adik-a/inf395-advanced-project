# QWork

A freelance marketplace platform for connecting clients and freelancers in Kazakhstan.

## Problem Statement

Freelancers in Kazakhstan often face challenges finding reliable clients, while businesses struggle to hire skilled professionals quickly. Existing global platforms may not fully support local needs such as language, payment systems, and regional market demands.

QWork aims to solve this by providing a localized freelance platform tailored specifically for the Kazakhstan market.

## Features

* User registration and authentication
* Job posting and project management
* Freelancer profile creation with skills and portfolio
* Search and filtering system for jobs and freelancers
* Secure communication between users
* Scalable backend for handling multiple users

## Installation

1. Clone the repository:
   git clone https://github.com/Beknur0/inf395-advanced-project.git

2. Navigate to the project folder:
   cd backend

3. Install dependencies:
   pip install -r requirements.txt

4. Run the application:
   uvicorn main:app --reload

## Usage

* Clients can create accounts and post jobs
* Freelancers can browse jobs and apply
* Users can communicate and collaborate on projects
* Admin can manage platform activity (if implemented)
